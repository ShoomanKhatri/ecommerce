# GadgetStore E-commerce Project

## Description
GadgetStore is an e-commerce platform built using the MERN stack. Users can browse products, manage their orders, and make payments through eSewa.

## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js
- Redux
- eSewa API
- Tailwind CSS
- React Toastify
- React Router

## Installation Instructions

1. download node package (https://nodejs.org/en)
2. open vs code terminal (ctrl + `)
3. in main file type (npm install) enter
4. change to frontend(cd .\frontend) type (npm i) enter
4. then come to main (cd ..)
5. type (cd .\backend) enter then type (npm i) enter

## Running project
1. in main GadgetStore type (cd .\frontend) then type (npm run dev) enter (terminal 1)
2. new terminal by clicking plus(+) button above type (cd .\backend) then type (npm run dev or node index.js) enter (terminal 2)
3. in terminal 1 (ctrl + click) in link or open localhost in browser if any (unexpected error) open link of (terminal 2)
4. After this project is runned
